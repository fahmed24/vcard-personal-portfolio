Live Infrastructure

This is a fully operational, production-grade Kubernetes environment—not a sandbox or a mock. Every service runs continuously, the cluster self-heals via GitOps, and this portfolio is built, deployed, and served directly from it. The full stack is infrastructure-as-code: no manual configuration, no one-off commands.

Architecture Overview

Hardware & Cluster

  • — 3-node Proxmox VE bare-metal cluster (proxmox1, proxmox2, proxmox3)
  • — k3s HA Kubernetes — 3 control-plane nodes + 6 workers, external PostgreSQL datastore
  • — HAProxy + Keepalived active/passive VIP for cluster API high availability
  • — VLAN-segmented network: k3s (VLAN 100), database (VLAN 200), load balancers (VLAN 50), LAN (VLAN 1)
  • — NVIDIA GPU node with passthrough — dedicated for LLM inference and media transcoding
  • — NFS-backed persistent storage, MinIO for S3-compatible object storage
  • — Daily VM and container backups via Proxmox Backup Server

Provisioning Pipeline

  • — Packer builds Ubuntu 22.04 golden images with cloud-init baked in
  • — Terraform provisions every VM and LXC container across all 3 Proxmox nodes
  • — Ansible bootstraps k3s servers and agents, load balancers, and database
  • — ArgoCD takes ownership from there — every application deployment is declarative and Git-driven
  • — No node in the cluster was configured manually

Running Services

  • Platform
  • — Traefik — ingress with automatic TLS termination via cert-manager (Let's Encrypt)
  • — ArgoCD — GitOps controller with auto-sync, pruning, and self-healing across all apps
  • Security & Identity
  • — Keycloak — SSO/OIDC provider for all services; no fallback password auth anywhere
  • — OAuth2 Proxy — group-based access enforcement at the ingress layer
  • — HashiCorp Vault + Vault Secrets Operator — all secrets stored in Vault, injected into pods; nothing sensitive in Git
  • Observability
  • — Prometheus — metrics collection across all cluster workloads
  • — Loki — log aggregation, stored in MinIO
  • — Tempo — distributed tracing via OTLP, stored in MinIO
  • — Grafana — unified observability dashboard with Keycloak SSO login
  • — Alloy — telemetry collection and forwarding agent
  • AI & Workloads
  • — Ollama — local LLM inference on NVIDIA GPU (gemma3, qwen2.5-coder:14b)
  • — Open WebUI — self-hosted chat interface, SSO-protected via Keycloak
  • — Jellyfin — hardware-accelerated media server with GPU transcoding

CI/CD — How This Page Got Here

Every change to this portfolio triggers a fully automated deployment — no manual steps:

  • — Push to GitHub → Actions builds and pushes a container image to GHCR
  • — Actions fires a repository_dispatch event to the homelab repo
  • — Homelab Actions updates the image tag in the Kubernetes manifest and commits
  • — ArgoCD detects the diff and auto-syncs — new pod is live within seconds
